# Sachin A — Cybersecurity Portfolio

Premium dark-themed portfolio website built with HTML5, CSS3, and Vanilla JavaScript.

## 📁 Folder Structure

```
Portfolio/
├── index.html        ← Main page
├── style.css         ← All styles (dark/light theme, responsive)
├── script.js         ← All JS (particles, typed, AOS, form, etc.)
├── robots.txt        ← SEO
├── sitemap.xml       ← SEO
├── images/
│   └── profile.jpg   ← Add your photo here (recommend 400×400 square)
└── assets/
    └── resume.pdf    ← Add your resume PDF here
```

## 🚀 Deploy to GitHub Pages

1. Push this folder to a GitHub repo (e.g. `username.github.io`)
2. Go to **Settings → Pages → Source: main / root**
3. Your site is live at `https://username.github.io`

## 🚀 Deploy to Vercel

1. `npm i -g vercel`
2. `vercel` in this folder
3. Follow prompts → live in 30 seconds

## ✏️ Personalise

| What to change | Where |
|---|---|
| Your photo | Replace `images/profile.jpg` |
| Your resume | Replace `assets/resume.pdf` |
| GitHub / LinkedIn URLs | Search `https://github.com/sachin` in `index.html` |
| Email / Phone | Contact section in `index.html` |
| CGPA, stats | `data-target` attributes on `.stat-num` in `index.html` |
| Project descriptions | `.project-card` blocks in `index.html` |

## 🛠 Tech Stack

- HTML5 + Semantic markup
- CSS3 — CSS custom properties, glassmorphism, grid, flexbox
- Vanilla JavaScript (no frameworks)
- [Particles.js](https://vincentgarreau.com/particles.js/) — background
- [Typed.js](https://mattboldt.com/demos/typed-js/) — typing animation
- [AOS](https://michalsnik.github.io/aos/) — scroll animations
- [Font Awesome 6](https://fontawesome.com/) — icons
- [Poppins + JetBrains Mono](https://fonts.google.com/) — typography
